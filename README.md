 ## 🧾 Project Overview

This project focuses on analyzing an **Employee Dataset using Microsoft Excel** to extract meaningful insights from raw HR data. The dataset contains **1000 employee records** with attributes such as Job Title, Department, Business Unit, Gender, Age, Hire Date, Salary, Bonus %, Country, City, and Exit Date.

## 🎯 Project Objective

The main goal of this project is to **clean, organize, and analyze employee data** to understand workforce distribution, salary patterns, and hiring trends.

## 🛠️ Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Sorting & Filtering
* Basic Data Analysis

## 🔍 Key Analysis Performed

* Cleaned and structured the raw employee dataset
* Handled missing values and corrected data formats
* Converted date columns into proper date format
* Analyzed employee distribution by **department and gender**
* Compared **salary and bonus percentage** across different job roles
* Explored **hiring trends and employee age distribution**

## 📈 Insights Generated

* Department-wise employee distribution
* Gender-based workforce analysis
* Salary comparison across job roles
* Hiring trends over different years
* Location-based employee analysis

## 🚀 Project Outcome

This project demonstrates how **Excel can transform raw HR data into meaningful insights**. It also helped strengthen my skills in **data cleaning, pivot table analysis, and data interpretation**, which are essential for data analysis tasks.
